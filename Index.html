FAQ — Pertanyaan Umum
📋 Bagaimana cara order?
+
Pilih produk yang kamu mau → klik Order & Bayar → isi nominal & nama → lanjut ke pembayaran → scan QRIS atau transfer ke DANA → upload bukti pembayaran → klik Kirim. Pesananmu langsung masuk ke admin! 🚀
⏱️ Berapa lama proses pesanan?
+
Setelah bukti pembayaran dikirim, admin memproses dalam 5–15 menit (jam aktif). Untuk top up game otomatis masuk dalam 5–30 menit. Jika lebih dari 1 jam, langsung hubungi admin via WhatsApp ya kak! 😊
🛡️ Apakah transaksi aman?
+
100% aman! LcbbbZstore sudah beroperasi sejak 2024 dengan ribuan customer puas. Pembayaran via QRIS resmi dan DANA yang terenkripsi. Data kamu tidak akan disalahgunakan. ✅
💸 Apakah bisa refund?
+
Ya! Kami memberikan garansi refund jika layanan tidak sesuai atau gagal diproses. Hubungi admin via WhatsApp dengan menyertakan Order ID kamu. Refund diproses dalam 1×24 jam kerja.
📦 Bagaimana jika pesanan belum masuk?
+
Jika pesanan belum masuk lebih dari 30 menit setelah pembayaran, segera hubungi admin via WhatsApp 085773286241 dengan menyertakan bukti pembayaran dan Order ID. Admin siap 24 jam! 💬
